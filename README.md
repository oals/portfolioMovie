# 프로젝트를 통해 느낀 점과 소감


이번 프로젝트는 제작기간이 짧았던 만큼 많은 기능을 넣지 못했다. <BR>
그럼에도 이 프로젝트가 나에게 의미 있었던 이유는 학원이 아닌 스프링 부트 책을 통해서 <BR> 배운 DTO TO ENTTIY 혹은 ENTTITY TO DTO 등의 정석적인 코드 구조를 실험 해볼 수 있어서이다. <BR>
