# 프로젝트를 통해 느낀 점과 소감

처음으로 안드로이드 어플을 만들어보았다. <BR>
안드로이드를 거의 모르는 상태에서 공부해가며 만들었기 때문에 시간도 오래 걸렸고 코드도 정석과는 거리가 멀어 보인다. <BR>
역시 책만 보는 것 보다는 직접 하나하나 만들어가면서 공부하는 게 안드로이드를 이해 하는데에 있어 큰 도움이 되는 것 같다. <BR>
